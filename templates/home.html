Welcome to Personal Finance Tracker

Track your income, expenses, and manage your finances with ease!

Concepts Used:

Python (Flask):
  • Handles routing, form submissions, and page rendering.
  • Manages backend logic for user registration, income/expense entry, and balance calculation.
  • Connects to MySQL database using mysql-connector.
QUERY:
  • Stores user data and transaction details in structured tables (users, transactions).
  • Performs SQL queries to insert, retrieve, and aggregate transaction data.
XAMPP:
  • Provides a local server environment to run MySQL.
  • Simplifies MySQL database setup and access through phpMyAdmin.
HTML & CSS:
  • Structures and styles the web pages.
  • Includes form inputs, tables, headings, and navigation bar.
  • Uses CSS for custom styling (e.g., background images, hover effects, layout design).
Finance Logic:
  • Distinguishes between income (entry) and expenses (withdraw).
  • Calculates total balance per user by aggregating transaction types.
